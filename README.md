# 🌶️ Spice Delight

A modern and responsive restaurant website built using **HTML, CSS, and JavaScript**. Spice Delight showcases authentic Indian cuisine with an interactive menu, chef's special dishes, and an easy-to-use ordering system.

## 🚀 Features

- Responsive Navigation Bar
- Mobile-Friendly Hamburger Menu
- Interactive Menu Search
- Category Filter Tabs
- Special Dishes Showcase
- Dynamic Order Calculation
- Quantity Management
- Form Validation
- Clean & Modern UI Design
